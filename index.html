Welcome to Islog Github pages.

This is the organization pages for Islog. We are a company specialized in RFID technology.

Our main open source projects

  • LibLogicalAccess is a rather complete RFID library supporting a wide range of devices and cards. The library is stable and used in production.
  • Leosac is an open source physical access control system designed to work on embedded hardware (Raspberry Pi). This project is under development and is not yet ready for use in production system.

And also...

  • Tempos which is a complete management system for rooms and bookings which can be coupled to a physical access control.
  • DokanNFC is as Dokan plugin that let the user mount an RFID card as a Windows filesystem. This project is under development.
